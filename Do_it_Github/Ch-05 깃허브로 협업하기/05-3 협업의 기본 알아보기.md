# 05-3 협업의 기본 알아보기

깃허브를 통한 협업에는 미리 여러 가지 약속을 정하고 시작한다.

- 브랜치 이름은 어떻게 정할 것인지
- 레이블은 어던 것을 사용할 것인지
- 이슈 관리는 어떻게 할 것인지
... 등등

어떤 식으로 협업 과정이 진행되는지에 대해 알아본다.
